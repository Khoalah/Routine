PUSH WORKOUTS

9 Exercises

  1. Incline bench press(smith)

    5-6 sets 8-10 reps (last set till failure)



  2. Chest press machine

    4 sets reps in each set till failure



  3. Cable (Upper chest)

    3 sets reps in each set till failure



  4. Shoulder press machine

    6 sets (first 4 sets high weight) decrease the reps from set 1 to 4 and then increase from 5 to 6



  5. Dumbell shoulder press

    4 sets heavy weight 10-12 reps



  6. Front raises

    3 sets dropset



  7. Overhead dumbell one arm Triceps extensions

    4 sets low weight 10-15 reps



  8. Triceps push down cable

    5 sets (1st set low weight and keep on increasing the weight till 4 and 5th is low weight) 8-12 reps 5th set reps till failure



  9. Triceps kickbacks (cable or dumbells)

    3 sets low weight reps till failure for each set



  10. Cardio
